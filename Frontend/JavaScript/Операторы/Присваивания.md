Операторы присваивания (`=`, `+=`, `-=`, `*=`, `/=`, `%=`, `**=`, `&=`, `|=`, `^=`, `<<=`, `>>=`, `>>>=`) присваивают значение переменной.

```ts
let a = 5;

a += 3; // 8
a -= 2; // 6
a *= 2; // 12
a /= 3; // 4
a %= 3; // 1
a **= 2; // 1
a &= 3; // 1
a |= 2; // 3
a ^= 1; // 2
a <<= 1; // 4
a >>= 1; // 2
a >>>= 1; // 1
```
